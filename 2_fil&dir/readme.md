### `pwd`
Prints the current working directory.

### `ls`
Lists files and directories.

Example:
```
ls -la
```

### `cd`
Changing Directory

Example:
```
cd /path/to/folder
cd /home/jarvis
```

### `mkdir`
Create Directory

Example:
```
mkdir my_folder
```

### `rm`
Remove files or directories

Example:
```
rm file.txt
rm -r folder
```
